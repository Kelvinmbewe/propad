# Spacing

Aurora spacing scale follows an 8px rhythm. Pair tokens to maintain consistent breathing room.

| Token | Value | Usage |
| --- | --- | --- |
| xs | 4px | Hairline dividers, tag paddings |
| sm | 8px | Inline gaps |
| md | 16px | Card padding, grid gutters |
| lg | 24px | Section spacing |
| xl | 32px | Page-level margins |

> Minimum layout padding: 16px (mobile), 24px (desktop).
